# Lab1: 나만의 시작 홈페이지 만들기

Gemini 웹 UI 챗팅창에서 각 단계의 프롬프트를 순차적으로 복사하여 실행하세요.

---

### 1단계: 기본 구조 생성
```text
오늘의 날씨와 주요 뉴스를 보여 주는 나만의 시작 홈페이지를 만들고 싶어
```

### 2단계: 기능 확장 및 테마 설정
```text
시계와 검색바도 추가하고 기술 관련 뉴스를 보여 줘. 밝고 깔끔한 스타일을 선호해.
```

### 3단계: 디자인 스타일 수정
```text
구글 스타일로 디자인을 수정해 줘
```

### 4단계: 할 일 목록 추가
```text
오늘의 할 일 목록을 추가해 줘
```

### 5단계: 레이아웃 및 카드 디자인 적용
```text
[현재 시간], [오늘의 할일], [오늘의 날씨], [최신 기술 뉴스] 아이템을 한 열에 두 개씩 배치하고, 메모지 안에 각각 들어있는 디자인으로 수정해 줘.
```
