# 손글씨 인식 프로그램 만들기
```text
MNIST 데이터셋을 사용해서 0부터 9까지의 손글씨 숫자를 인식하는 인공지능 프로그램을 만들고 싶어. mnist_app.py라는 파일로 만들어줘. 
```

```text
지금 만든 모델을 사용해서, 마우스로 직접 숫자를 그릴 수 있는 윈도우 창(UI)을 띄워줘. 내가 숫자를 그리고 '예측'버튼을 누르면 AI가 어떤 숫자인지 알려주게 해줘
```

```text
숫자 인식 프로그램을 윈도우에서 바로 실행할 수 있도록 만들어줘
```

```text
GEMINI.md 파일에 '향후 계획' 섹션을 추가해서 '웹 브라우저에서 실행 가능한 버전 만들기'로 추가해줘
```

```text
# 앞으로 만드는 모든 파이썬 파일 상단에는 생성 날짜와 시간을 주석으로 표시해 줘. 그리고 한글 깨짐 방지 코드는 항상 자동으로 포함해 줘.
```

```text
손 글씨 인식 프로그램을 웹 버전과 데스크톱 버전으로 나누어 개발하고 싶어. web_version과 desktop_version 폴더를 만들고, 각 폴더에 해당하는 GEMINI.md 파일도 생성해줘.
```

```text
웹 버전 프로그램을 브라우저에서 실행해 줘
```
