#### 레시피추천 앱 제작 (3단계)

모델은 변경가능합니다.
```
앞서 만든 OPENAI API를 이용해서, 냉장고 사진에서 재료를 인식하고 레시피를 추천하는 웹 애플리케이션을 만들고 싶어. 다음과 같이 3단계로 나누어서 PRD를 만들어 줘.
1단계
- 사용자 로그인 후 이미지를 입력받고 gpt-4o-mini 모델을 사용해서 이미지를 인식해. 
- 사용자는 ID, 암호, 이름, email 주소로 가입할 수 있는데 현재는 로컬 파일에 사용자 정보를 저장할거야.
- 로그인테스트를 위해 test/1234인 사용자를 생성해줘.
2단계
- 1단계에서 얻은 정보와 gpt-4o-mini 모델을 사용해서 레시피를 생성해.

3단계
- 사용자 프로필을 만들어서 레시피를 저장해.
- MySQL 데이터베이스에 연결해서 회원가입, 로그인을 처리하게해.
- MySQL 데이터베이스 정보는 다음과 같아. 
  host : 43.201.66.114 
  user/pw : test/Mydb1234!
  DB : testdb
  Table : member
  Table 구조 및 예제 데이터
mysql> SELECT * FROM member;
+--------+----------+-----------+----------------------+
| userid | password | username  | email                |
+--------+----------+-----------+----------------------+
| test   | vibe1234 | 테스트    | testuser@example.com |
+--------+----------+-----------+----------------------+



각 단계를 PRD_step1.md, PRD_step2.md, PRD_step3.md로 나누어 저장해 줘
```
