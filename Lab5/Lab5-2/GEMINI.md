# GEMINI.md

이 파일은 이 저장소의 코드로 작업할 때 Gemini CLI에 대한 지침을 제공합니다.

## 저장소 상태

이 저장소는 VibeCoding 워크스페이스에서 새로 초기화된 저장소(Study-03)입니다. 현재 저장소는 비어 있으며 첫 번째 프로젝트 구현을 기다리고 있습니다.

## 관련 프로젝트

이 저장소는 일련의 학습 프로젝트 중 일부입니다:

* Study-01: 애플리케이션의 데스크톱 및 웹 버전을 포함함
* Study-02: HTML, CSS, JavaScript를 사용한 웹 기반 할 일(todo) 애플리케이션을 포함함

## 프로젝트 설정

이 저장소는 비어 있으므로, 첫 번째 구현 요청에 따라 프로젝트 유형과 구조가 결정됩니다. 여기서 생성될 수 있는 일반적인 프로젝트 유형은 다음과 같습니다:

* **웹 애플리케이션**: Study-02와 유사한 HTML/CSS/JavaScript 프로젝트
* **데스크톱 애플리케이션**: 잠재적으로 Study-01의 프레임워크를 사용
* **풀스택 애플리케이션**: 프런트엔드 인터페이스와 백엔드 API
* **기타**: 특정 요구 사항에 기반함

## 초기 개발 작업

이 저장소에서 새 프로젝트를 시작할 때:

1. 사용자로부터 프로젝트 유형 및 요구 사항을 확인합니다.
2. 프로젝트 유형에 따라 적절한 디렉토리 구조를 생성합니다.
3. 필요한 경우 패키지 관리(npm, pip 등)를 초기화합니다.
4. 기본 구성 파일(.gitignore 등)을 설정합니다.
5. 요청에 따라 핵심 기능을 구현합니다.

## Git 구성

저장소는 git으로 초기화되었습니다. 다음 사항을 기억하세요:

* 명확한 메시지와 함께 의미 있는 커밋을 작성하세요.
* 프로젝트 유형에 적합한 .gitignore 패턴을 사용하세요.
* 사용자가 명시적으로 요청한 경우에만 커밋하세요.
